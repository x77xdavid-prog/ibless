# 아이블레스 (iBless) — 영어학원 랜딩 페이지

유아·초등·중등·고등 대상 **수준별 1:1 맞춤 영어 학습 브랜드 "아이블레스"** 의
정적(static) 랜딩 페이지입니다. 의존성 없이 브라우저에서 바로 동작합니다.

> **제작 배경**: 잘 만들어진 영어학원 사이트의 **레이아웃·섹션 구성·디자인 시스템을 레퍼런스로 참고**하여,
> 문구·브랜딩·이미지는 모두 **아이블레스 자체 오리지널**로 새로 작성했습니다.
> 타사의 로고/사진/마케팅 문구를 그대로 사용하지 않았습니다.

---

## 파일 구조

```
아이블레스/
├── index.html        # 페이지 마크업 (시맨틱 HTML)
├── css/
│   ├── tokens.css    # 디자인 토큰 (색상·타이포·간격·모션 변수)
│   └── main.css      # 레이아웃 & 컴포넌트 스타일
├── js/
│   └── main.js       # 탭 전환·모바일 메뉴·스크롤 리빌·카운트업
└── assets/           # (이미지 교체용 폴더)
```

## 실행 방법

`file://` 직접 열기 대신 간단한 로컬 서버 사용을 권장합니다.

```bash
# 프로젝트 폴더에서
python -m http.server 8777
# 브라우저에서 http://127.0.0.1:8777 접속
```

| 1 | 헤더 | 로고 · GNB · 학습 사이트 · 무료 레벨테스트 (모바일 햄버거) |

| — 인증평가 | 테스트 로드맵 + 리포트 샘플 (옐로) |

| 6 | 푸터 | 약관·개인정보·사업자 정보 |

## 커스터마이징 가이드

### 1. 색상 / 폰트 / 간격
`css/tokens.css` 의 CSS 변수만 바꾸면 전체 톤이 바뀝니다.

```css
--color-primary: #ff7a00;  /* 메인 주황 */
--color-sky:     #00b8e4;  /* 커리큘럼 시안 */
--color-yellow:  #feca0a;  /* 인증평가 옐로 */
--font-display:  "Jua";    /* 로고·강조 폰트 */
```

### 2. 교체해야 할 플레이스홀더 (운영 전 필수)
- **이미지**: `이미지`, `리포트 이미지` 라벨이 표시된 회색 박스
  (`.ph-thumb`, `.ph-doc`) → 실제 상품/리포트 이미지로 교체
- **푸터 사업자 정보**: 상호·대표·사업자등록번호(477-29-02140)·주소 반영 완료.
  남은 항목은 **대표전화 `(전화번호 입력)`** 만 실제 번호로 교체하면 됩니다.
- **링크(`#` 앵커)**: 각 메뉴/버튼의 `href="#..."` → 실제 페이지 경로로 연결
- **스탯 수치**: 히어로 카드의 학습률·연속일수, 스탯 밴드 수치 등은 예시값

### 3. 교재/단계명
커리큘럼 단계명(Phonics·Basic·Master·Advanced)과 교재명·대상 시험은 예시입니다.
실제 아이블레스 커리큘럼에 맞게 `index.html` 의 해당 섹션을 수정하세요.

## 접근성 / 품질

- 시맨틱 태그(`header/nav/main/section/footer`) + ARIA 탭 패턴
- 키보드 포커스 링, `skip-link`(본문 바로가기)
- `prefers-reduced-motion` 존중 (모션 최소화 시 애니메이션 비활성)
- 반응형: 1440 / 1024 / 768 / 480 / 375 검증 완료
- **JS 미동작 시에도 본문이 모두 노출**되도록 프로그레시브 인핸스먼트 적용
- 이모지 대신 인라인 SVG 아이콘 사용
